# Screenshots for Blog Posts

Place your screenshots in this directory with the following filenames:

1. **claude-code-welcome.png** - Screenshot of Claude Code welcome screen showing the project setup
2. **chrome-extensions-page.png** - Screenshot of Chrome Extensions management page showing "Load unpacked" button
3. **readtime-extension-popup.png** - Screenshot of the ReadTime extension popup UI (optional, if you have a separate popup screenshot)
4. **readtime-in-action.png** - Screenshot of ReadTime extension showing reading time on a blog post (the one with the extension overlay)

The blog post will automatically reference these images when they're placed in this directory.
